# Semester Manager

A simple web app that helps students track their courses, units, grades, and CGPA across a semester.

Built to solve a real problem: keeping academic records
organised without spreadsheets or paper notes.

---

## Features

### Core Features
- Add and manage courses for a semester
- Automatic CGPA calculation
- Total courses and total units summary
- Grade distribution visualized with a pie chart
- Course limit enforcement (prevents adding more than allowed)
- Light / Dark mode toggle
- Persistent data using local storage

### Optional / Nice-to-Have
- Responsive layout for mobile and desktop
- clean UI with clear visual hierarchy
- Simple and distraction-free workflow
